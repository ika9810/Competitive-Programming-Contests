Problem Statement

You are given an NNN-by-NNN matrix A=(ai,j)A=(a_{i,j})A=(ai,j​), where ai,j∈{0,1}a_{i,j} \in \{0,1\}ai,j​∈{0,1}.

We have the following directed graph.

  • The graph has NKNKNK vertices numbered 1,2,…,NK1,2,\ldots,NK1,2,…,NK.
  • The edges correspond to the NKNKNK-by-NKNKNK matrix X=(xi,j)X=(x_{i,j})X=(xi,j​) obtained by arranging K2K^2K2 copies of AAA in KKK rows and KKK columns (see Sample Input/Output 1 for an example). If xi,j=1x_{i,j}=1xi,j​=1, there is a directed edge from vertex iii to vertex jjj; if xi,j=0x_{i,j}=0xi,j​=0, that edge does not exist.

Answer the following question for i=1,2,…,Qi=1,2,\ldots,Qi=1,2,…,Q.

  • Find the shortest length (number of edges) of a path from vertex sis_isi​ to vertex tit_iti​. If there is no such path, print -1 instead.

  • si≠tis_i \neq t_isi​=ti​
  • All values in the input are integers.
